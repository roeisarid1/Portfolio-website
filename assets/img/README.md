# Images Directory

This folder contains images and icons for the portfolio website.

## Required Images

### 1. Profile/Avatar Image

- **Filename**: `avatar-placeholder.jpg`
- **Recommended size**: 300x300px or larger (square format)
- **Format**: JPG, PNG, or WebP
- **Usage**: Displayed in the About section
- **Description**: Your professional headshot or avatar

## Optional Images

You can add additional images for:

- Project screenshots
- Background images
- Icons
- Logos

## Image Optimization Tips

1. **Compress images** before uploading to reduce file size
   - Use tools like TinyPNG, ImageOptim, or Squoosh
2. **Use appropriate formats**:
   - JPG for photos
   - PNG for images with transparency
   - WebP for modern browsers (better compression)
3. **Recommended dimensions**:
   - Profile image: 300x300px to 500x500px
   - Project screenshots: 800x600px to 1200x900px
4. **Keep file sizes small**:
   - Profile image: < 200KB
   - Other images: < 500KB each
